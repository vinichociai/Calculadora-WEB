# Calculadora-WEB

Projeto web de calculadora desenvolvido com **HTML, CSS e JavaScript**, criado para praticar lógica de programação, manipulação do DOM e construção de interface interativa no navegador.

## Sobre o projeto

Este projeto foi desenvolvido como estudo prático de front-end, com foco na criação de uma calculadora funcional utilizando tecnologias básicas da web.

A aplicação permite realizar operações matemáticas simples por meio de uma interface visual, trabalhando eventos de clique, atualização dinâmica da tela e organização dos arquivos do projeto.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Interface visual de calculadora
* Operações matemáticas básicas
* Manipulação do DOM com JavaScript
* Eventos de clique nos botões
* Organização de arquivos HTML, CSS e JS

## Estrutura do projeto


Calculadora-WEB/

├── styles.css

## Como executar

Clone o repositório:


git clone https://github.com/vinichociai/Calculadora-WEB.git


Acesse a pasta do projeto:


cd Calculadora-WEB


Abra o arquivo `index.html` no navegador.
